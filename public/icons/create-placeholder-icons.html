PWA Icon Generator for JobQuest

Este archivo HTML puede usarse para generar iconos placeholder.

Para generar los iconos reales, necesitas:

  1. Un icono fuente 512x512px con el logo de JobQuest (diamond)
  2. Usar el script: ./generate-icons.sh icon-source.png
  3. O usar una herramienta online como: https://realfavicongenerator.net/

Los iconos deben colocarse en: public/icons/
